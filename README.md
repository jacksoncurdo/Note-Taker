# Note-Taker

## Project Description
An application called Note Taker that can be used to write and save notes.

## Project Motivation
To have a fully functioning application that can write and save notes using node.js and Express.

## Installation 
- npm init

- npm install node

- npm install express

- heroku create


## Technologies used 
- Express
- Node.js
- jquery 
- json 
- HTML 
- CSS 
- Javascript
